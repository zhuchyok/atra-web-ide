## **🔧 ТЕХНИЧЕСКИЕ ИЗМЕНЕНИЯ**

### **Файлы изменены:**
- `signal_live.py` - обновлены форматы всех типов сигналов

### **Обновленные функции:**
- Формирование новых торговых сигналов
- Формирование DCA сигналов
- Формирование накопленных DCA сигналов

### **Ключевые изменения:**
1. **Удалены строки с командами** - убраны все `/accept` команды из сообщений
2. **Упрощена клавиатура** - оставлена только кнопка "Принять"
3. **Переход на Markdown** - заменены HTML теги на Markdown форматирование
4. **Сохранен полный технический анализ** - все индикаторы остались
