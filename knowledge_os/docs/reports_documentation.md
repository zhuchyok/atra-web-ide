# PiuX_Trade: Документация по отчётам

## Описание

В этом модуле описаны все возможности по генерации, анализу и отправке отчётов в системе PiuX_Trade.

---

## Пример отчёта

```
[PiuX_Trade][REPORT] Генерация отчёта...
[PiuX_Trade][REPORT] Отправка отчёта...
```

---

## Возможности
- Генерация отчётов по сделкам, портфелю, equity curve
- Отправка отчётов в Telegram и на email с фирменным стилем PiuX_Trade
- Визуализация результатов

---

## Использование

Вызовите функцию:

```
generate_report(...)
send_report(...)
```
