```python
def calculate_dynamic_tolerance(df, i):
    """
    –†–∞—Å—Å—á–∏—Ç—ã–≤–∞–µ—Ç –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏–π tolerance –Ω–∞ –æ—Å–Ω–æ–≤–µ –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç–∏
    """
    # –†–∞—Å—Å—á–∏—Ç—ã–≤–∞–µ–º –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å –∑–∞ –ø–æ—Å–ª–µ–¥–Ω–∏–µ 20 —Å–≤–µ—á–µ–π

    volatility = closes.std() / closes.mean()

    # –ë–∞–∑–æ–≤—ã–π tolerance
    base_tolerance = 0.003  # 0.3%

    # –ê–¥–∞–ø—Ç–∏—Ä—É–µ–º –ø–æ–¥ –≤–æ–ª–∞—Ç–∏–ª—å–Ω–æ—Å—Ç—å
    dynamic_tolerance = base_tolerance * (1 + volatility * 10)

    dynamic_tolerance = max(0.001, min(dynamic_tolerance, 0.02))

    return dynamic_tolerance
```
