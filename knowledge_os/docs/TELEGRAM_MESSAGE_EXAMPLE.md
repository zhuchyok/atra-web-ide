## 📋 Как это выглядит в Telegram:

### ✅ Преимущества Markdown формата:

1. **Копирование значений:**
   - При нажатии на `FILUSDT` → копируется `FILUSDT` (без кавычек)
   - При нажатии на `2.193` → копируется `2.193` (без кавычек)
   - При нажатии на `1.9x` → копируется `1.9x` (без кавычек)

2. **Читаемость:**
   - Нет HTML тегов (`<code>`, `<b>`)
   - Чистый текст с эмодзи
   - Легко читается и копируется

3. **Форматирование:**
   - Обратные кавычки `` ` `` выделяют важные значения
   - Эмодзи улучшают визуальное восприятие
   - Компактный и информативный формат

---

## 🔄 Сравнение форматов:

### ❌ Было (HTML):
```
📊 Символ: <b><code>FILUSDT</code></b>
💰 Цена входа: <code>2.193</code>
```

**Проблемы:**
- Видны HTML теги
- Сложно копировать
- Неудобно читать

### ✅ Стало (Markdown):
```
📊 Символ: `FILUSDT`
💰 Цена входа: `2.193`
```

**Преимущества:**
- Нет тегов
- Легко копировать
- Удобно читать
