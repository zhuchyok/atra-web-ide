# UI/UX Design Standards for Singularity

## 1. General Principles
- **Clarity**: The interface should be intuitive and self-explanatory.
- **Consistency**: Use consistent colors, fonts, and spacing across all pages.
- **Responsiveness**: Layout must adapt gracefully to different screen sizes.
- **Accessibility**: High contrast, readable font sizes (min 14px), and keyboard navigability.

## 2. Visual Hierarchy
- Use font weight and size to distinguish headings (H1, H2, H3).
- Primary actions should be visually distinct (e.g., solid buttons).
- Secondary actions should be less prominent (e.g., outlined buttons or text links).

## 3. Spacing and Alignment
- Use a standard grid system (8px or 10px base).
- Ensure consistent padding inside containers.
- Elements must be aligned to a common axis.

## 4. Color Palette
- **Primary**: Corporate Blue (#007AFF) or similar.
- **Success**: Green (#34C759).
- **Warning**: Orange (#FF9500).
- **Error**: Red (#FF3B30).
- **Background**: Neutral light (#F2F2F7) or dark (#1C1C1E).

## 5. Typography
- Use system fonts (San Francisco on Mac, Segoe UI on Windows).
- Maintain line height of 1.4 - 1.6 for body text.
