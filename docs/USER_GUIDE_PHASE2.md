# Руководство пользователя (Фаза 2)

## Новые возможности

### 1. Быстрые ответы в Чате

- **Приветствия и короткие реплики** («привет», «спасибо», «пока») обрабатываются без вызова LLM — ответ приходит сразу.
- **Фактические вопросы** («сколько стоит подписка?», «какой порт?») при наличии данных в базе знаний могут быть обработаны через RAG-light за 100–200 ms.
- **Сложные запросы** (анализ, сравнение, план) в режиме Чат по-прежнему идут через MLX/Ollama/Victoria; при этом может показываться рекомендация перейти в режим Агент.

### 2. Рекомендация перейти в Агент

- Система помечает запросы, для которых полезнее режим Агент (глубокий анализ, много шагов).
- В ответе в Чате может появиться шаг **«Рекомендация»** с текстом вроде: «Этот запрос требует глубокого анализа. Для наиболее полного ответа перейдите в режим «Агент». Продолжаю в текущем режиме…»
- Рядом отображается кнопка **«Перейти в Агент»** — по нажатию открывается вкладка «Агент».

### 3. Режимы: Чат, Агент, План

- **Чат** — быстрые ответы: шаблоны, RAG-light, при необходимости MLX/Ollama; плюс рекомендация Агента для сложных запросов.
- **Агент** — полный сценарий с Victoria (шаги, план, выполнение).
- **План** — только план без выполнения.

## Советы по использованию

### Для быстрых ответов

- Используйте режим **Чат** для приветствий и простых вопросов.
- Формулируйте фактические вопросы конкретно — так выше шанс ответа из базы знаний (RAG-light).

### Для сложного анализа

- Используйте режим **Агент** для глубокого анализа, сравнений, планов.
- Либо отправьте сложный запрос в Чате и при появлении рекомендации нажмите **«Перейти в Агент»**.

### Для планирования

- Используйте режим **План** для пошаговых инструкций без выполнения.
