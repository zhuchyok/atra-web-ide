## ФАЗА 1: ПРОИЗВОДИТЕЛЬНОСТЬ И ОПТИМИЗАЦИЯ

### Задача 1.1: Параллельная обработка запросов

**Цель:** Уменьшить latency на 40-60% через параллельную обработку.

**Файлы:**
- `knowledge_os/app/parallel_request_processor.py` (новый)
- `knowledge_os/app/ai_core.py` (интеграция)

**Детали реализации:**
1. Создать `ParallelRequestProcessor` класс с пулом воркеров
2. Использовать `asyncio.Semaphore` для ограничения параллелизма
3. Модифицировать `run_smart_agent_async` в `ai_core.py` для параллельных запросов к локальным и облачным моделям
4. Добавить балансировку нагрузки между моделями
5. Логирование метрик параллельной обработки

**Критерии успеха:**
- Latency p95 < 2 секунды
- Параллельная обработка до 5 запросов одновременно
- Улучшение throughput на 40-60%
