# Архив документов

Сюда переносятся одноразовые отчёты и статусы, которые были в корне репозитория, чтобы не засорять корень. Актуальная документация — в **docs/** (MASTER_REFERENCE, PROJECT_ARCHITECTURE_AND_GUIDE и др.).

## root_reports/

Файлы из корня репозитория (исторические отчёты, статусы, отчёты о завершении задач). Оставлены для истории; для текущей работы смотрите docs/MASTER_REFERENCE.md и связанные документы.

**В корне оставлены:** README.md, PLAN.md, VICTORIA.md, VERONICA.md, requirements.txt и конфиги/скрипты. (requirements.txt не переносился — остаётся в корне.)

## obsolete_backups/

Файлы с суффиксом .backup, перенесённые из исходников (например src/filters/manager.py.backup), чтобы не засорять рабочие каталоги. Текущий код — в src/; бэкапы сохранены для истории.
