# Эталон: «Покажи список файлов в корне / в директории»

**Цель (goal):** «покажи список файлов в корне проекта», «покажи файлы в frontend», «список файлов в …».

**Эталонный ответ:** вывод команды (ls / dir) или структурированный список; при делегировании в Veronica — STDOUT команды (в ответе бывают STDOUT и total от вывода ls). Без лишних комментариев сверх необходимого; при большом выводе — можно ограничить или сгруппировать.

**Критерии:** пользователь получает актуальный список; маршрут может быть Veronica (task_type: veronica) для простого «покажи файлы».
