# Эталон: «Напиши одну строку кода / пример кода»

**Цель (goal):** «напиши одну строку кода на Python: вывод текущей даты», «пример кода для …».

**Эталонный ответ:** код в блоке (markdown ```), рабочий и по заданию. Минимум пояснения; без длинных отступов «эта строка точно выводит…» — достаточно блока кода и при необходимости одной строки пояснения.

**Критерии:** код выполняет запрос; формат — блок кода; метод может быть ReAct (method: react).
