# Эталон: «Какой статус проекта?»

**Цель (goal):** запросы вида «какой статус проекта», «статус проекта», «как дела с проектом».

**Эталонный ответ:** Статус проекта смотрите в дашборде (Corporation Dashboard, порт 8501) и в списке задач Knowledge OS. Опираюсь на факты из MASTER_REFERENCE и задач, не придумываю сроки.

**Коротко для RAG/контекста (использовать при ответе):** «Статус проекта — в дашборде (порт 8501) и в задачах Knowledge OS. Детали в MASTER_REFERENCE.»

(Кратко и по факту: что сейчас в фокусе по MASTER_REFERENCE, дашбордам, задачам; при неизвестном статусе — предложить дашборд/задачи, не выдумывать сроки.)

**Критерии:** ответ опирается на данные (задачи, документация) или явно говорит «проверьте дашборд / список задач»; нет галлюцинаций про сроки и этапы.
